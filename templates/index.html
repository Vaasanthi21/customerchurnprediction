Customer Churn Prediction

Project Overview

This project focuses on predicting customer churn using machine learning techniques. The aim is to help businesses identify customers who are likely to discontinue their services, enabling proactive retention strategies.

Responsibilities

- Collected and preprocessed customer data including demographics, service usage, and account details.
- Applied feature engineering and exploratory data analysis to uncover churn patterns.
- Built and evaluated classification models using scikit-learn.
- Deployed the model as a Flask web application for interactive predictions.

Tools & Technologies

Python, Flask, Pandas, NumPy, Scikit-learn, Jinja2, Gunicorn, Render (deployment).

Outcome

The deployed application allows users to input customer attributes and receive churn predictions in real time. This project demonstrates end-to-end skills in data science, machine learning, and web deployment.
